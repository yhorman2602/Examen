# Inventario de Productos

Es una aplicación que sirve para el inventario de los productos de una tienda o cualquier taller que necesite registrar sus productos.

## Funcionalidades

- **Agregar productos**: Registra productos con nombre, categoría, precio, cantidad y descripción.
- **Editar productos**: Modifica los datos de cualquier producto registrado.
- **Eliminar productos**: Elimina productos con confirmación para evitar errores.
- **Buscar y filtrar**: Busca por nombre/descripción y filtra por categoría.
- **Ordenar**: Ordena la lista por nombre, categoría, precio o cantidad.
- **Estadísticas**: Visualiza el total de productos, unidades en stock, valor total del inventario y alertas de stock bajo.
- **Persistencia**: Los datos se guardan automáticamente en el navegador (localStorage).

## Categorías disponibles

Herramientas, Materiales, Electrónica, Ropa, Alimentos, Repuestos, Otros.

## Uso

Abre el archivo `index.html` en cualquier navegador moderno. No requiere instalación ni servidor.

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
